Memory Guard

A solution to backup the pictures from the mobile to the PC.

Download .zip for WindowsDownload .apk for AndroidView on Github

Welcome to Memory Guard

This application upload pictures from the mobile to the PC. You can install the APK on Andoid and install the PC build on Windows. Input the information between the devices to help them connect. When the devices are connected, the mobile will upload the files you selected to the PC. For more information, there is the more detailed instruction in the application. You can run it and view the documention in it.
